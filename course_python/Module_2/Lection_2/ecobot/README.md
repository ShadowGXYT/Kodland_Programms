# Eco Bot 🌱

Eco Bot is a Telegram bot that helps adults reduce waste and live more sustainably.

## Features
- 🌍 Random eco-friendly tips
- ♻️ Daily eco challenges
- 😊 Emojis for better user experience
- 🧹 Clear previous messages with /clear
- 🧩 Clean and modular code structure

## Technologies
- Python
- pyTelegramBotAPI (telebot)
- python-dotenv (for environment variables)

## Commands
/start – Start the bot  
/tip – Get an eco tip  
/challenge – Get a daily challenge  
/help – Show help  
/clear – Clear previous bot messages in chat

## Goal
Encourage environmentally friendly habits in everyday life.
